# *radnano* : cadnano 2.5 beta

`cadnano` looks better in lower case

## Running

to run with a GUI from the command line :

    cadnano

when in a python interpreter you can just:

    import cadnano

And script your designs without a GUI

## Install

    pip install cadnano
